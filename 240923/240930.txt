1. Authentication
- 회원가입 기능 구현
- 신규 생성된 회원정보 > 로그인 기능 구현
- sns 계정 연동 > 로그인 기능 구현

2. Firestore Database
-UI 화면을 통해서 사용자로부터 입력받은 DATA 활용 > 컬렉션 > 문서 > 빌드
컬렉션 : 데이터가 저장될 상위 카테고리
문서 : 새로운 컨텐츠가 저장(업로드) 될때마다 신규로 생성되는 카테고리(*고유한id값으로 생성)
필드: 컨텐츠가 신규로 생성 필요한 세부정도를 담고 있음(*컨텐츠가 생성된 날짜정보 // 컨텐츠 내용 // 해당 컨텐츠의 고유id // 컨텐츠를 생성한 사용자의 정보)

3. Sortage
- 이미지 혹은 동영상 물리적으로 업로드
- 컬렉션 : 이미지 혹은 동영상 컨텐츠가 저장될 상위 카테고리
- 문서 : 새로운 이미지 혹은 동영상 컨텐츠가 저장(업로드)될 때마다 신규로 생성되는 카테고리
- 필드 : 업로드한 이미지 혹은 동영상 세부 address 정보가 담긴 필드
